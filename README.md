# discogs-collector
Uses Discogs-API to collect information from your collection on Discogs.

## Installation

## Usage

Input: Discogs Token

Output: csv of all your tracks from your whole collection with following data

- Id
- Title
- Artist
- Comment ("Vinyl")
- Duration (millisec)
- (BPM)
- Genre
- Track
- Year
- Album
- Language
- Century
- Subgenre
- Discnumber
- Catalog
- Label
- Labelcode
- Date
- Media
